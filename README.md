# 🎉 Event Registration Portal

A modern and responsive **Event Registration Portal** built using **Flask, SQLite, HTML, CSS, and JavaScript**. The application enables users to register for events online while providing administrators with a secure dashboard to manage registrations and event details.

---

## 🌐 Live Demo

**Live Website:**  https://event-registration-portal-xozo.onrender.com

## ✨ Features

### User Features

* Register for events
* View event information
* Countdown timer
* Mobile responsive interface

### Admin Features

* Secure admin login
* View registrations
* Search registrations
* Delete registrations
* Export registrations to CSV

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Harika101126/event-registration-portal.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```
